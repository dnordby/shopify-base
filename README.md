### Base Shopify Theme

To initialize:
  - Clone the repo
  - `npm install`
  - Create a [`config.yml`](http://shopify.github.io/themekit/) file

Development:
 - Themekit and webpack can be run separately by running each of the following in separate tabs:
   - `theme watch`
   - `webpack --watch`

 - Or running the following in a single tab:
    - `npm run watch`
